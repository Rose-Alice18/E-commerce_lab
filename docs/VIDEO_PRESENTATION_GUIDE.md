## Technical Setup Guide

### Equipment Needed
- **Computer:** Your development machine with XAMPP running
- **Microphone:** Built-in mic is okay, external USB mic is better
- **Screen Recording Software:**
  - **Windows:** OBS Studio (free), Camtasia (paid), Loom (free)
  - **Mac:** QuickTime Player (free), ScreenFlow (paid)
- **Video Editing Software (optional):**
  - DaVinci Resolve (free)
  - iMovie (Mac, free)
  - Shotcut (free)
  - Adobe Premiere (paid)

### Recording Setup

#### Option 1: Screen Recording + Voiceover (Recommended)
1. Open OBS Studio or Loom
2. Set up screen capture (full screen or browser window)
3. Test audio levels (speak normally, check input meter)
4. Open PharmaVault in browser
5. Have your slides ready in another tab/window
6. Record entire presentation in one take (or segment by segment)

#### Option 2: Picture-in-Picture (Advanced)
1. Use OBS Studio
2. Add webcam source (small box in corner)
3. Add screen capture source
4. Arrange layout (webcam in bottom-right corner)
5. Record with both your face and screen visible

#### Option 3: Slides + Demo Hybrid
1. Record slides with voiceover (PowerPoint can do this)
2. Record separate screen demo
3. Combine in video editor
4. Add transitions between segments

### Recording Tips

**Before Recording:**
- [ ] Close unnecessary programs (to avoid notifications)
- [ ] Turn off phone/notifications
- [ ] Clear browser tabs (close email, social media)
- [ ] Test audio (record 30 seconds, play back)
- [ ] Prepare demo data (sample products, test accounts)
- [ ] Have a glass of water nearby
- [ ] Practice your script 2-3 times

**During Recording:**
- [ ] Speak clearly and at moderate pace
- [ ] Pause briefly between sections
- [ ] Use cursor to highlight important elements
- [ ] Zoom in on small text if needed
- [ ] Smile (it comes through in your voice)
- [ ] If you make a mistake, pause 5 seconds and restart that sentence

**After Recording:**
- [ ] Review entire video
- [ ] Check audio quality (no clipping, clear speech)
- [ ] Verify all sections are included
- [ ] Trim dead time at beginning/end
- [ ] Add title slide at start
- [ ] Export in high quality (1080p MP4)

### Video Specifications

**Technical Requirements:**
- **Format:** MP4 (H.264 codec)
- **Resolution:** 1920x1080 (1080p) or 1280x720 (720p minimum)
- **Frame Rate:** 30 fps
- **Audio:** 44.1 kHz, stereo
- **File Size:** Under 500 MB (compress if needed)
- **Duration:** 9-11 minutes (aim for 10 minutes)

**Export Settings (OBS Studio):**
```
Format: MP4
Encoder: x264
Bitrate: 2500 Kbps (for 1080p)
Audio Bitrate: 160 Kbps
```

## Troubleshooting

### Common Issues

**Issue: Video file too large**
- Solution: Compress using HandBrake (free tool)
- Settings: H.264, RF 23, 720p resolution

**Issue: Audio too quiet**
- Solution: Boost audio in Audacity (free tool)
- Or re-record closer to microphone

**Issue: Screen too small to read**
- Solution: Zoom in browser (Ctrl +)
- Or use OBS Studio to crop to relevant area

**Issue: Demo not working during recording**
- Solution: Test everything before recording
- Have backup screenshots ready

**Issue: YouTube processing stuck**
- Solution: Wait 30 minutes, refresh
- Or try different browser
- Or upload to Google Drive instead
